Solution

From the description we know that the algorithm in use is RSA, and that three moduli are used to encrypt three separate ciphertexts that each include a portion of the flag.

1
2
3

n1 = p*q
n2 = q*r
n3 = p*r

The plaintext cannot be retrieved without the values of p, q, and r; so, to obtain the values of p, q, and r, we determine the greatest common divisor between the three moduli, which are n1, n2, and n3.

1
2
3
4
5

from math import gcd

p = gcd(n1, n3)
q = gcd(n1, n2)
r = gcd(n2, n3)

Second step is to find the values of phi1, phi2, and phi3 for n1, n2, and n3 used to encrypt the three parts of the flag.

By definition, phi(n) is Euler’s totient function. It counts the number of integers less than n that have no shared factors with n.

The formula for calculating phi is phi(N) = (p - 1) * (q - 1)

Once we calculate the value of phi, we can also calculate d using the formula d = e^-1 mod phi(n)

from Crypto.Util.number import inverse

e = 65537

phi1 = (p - 1) * (q - 1)
d1 = inverse(e, phi1)

phi2 = (q - 1) * (r - 1)
d2 = inverse(e, phi2)

phi3 = (p - 1) * (r - 1)
d3 = inverse(e, phi3)

Now, to compute the three parts of the flag we simply calculate pt = c^d mod n.

from Crypto.Util.number import long_to_bytes

pt1 = pow(c1, d1, N1)
pt2 = pow(c2, d2, N2)
pt3 = pow(c3, d3, N3)

part1 = long_to_bytes(pt1).decode()
part2 = long_to_bytes(pt2).decode()
part3 = long_to_bytes(pt3).decode()

print(part1 + part2 + part3)
